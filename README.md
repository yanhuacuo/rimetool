# rimetool

中州韵助手（重构版）

适配，及兼容性问题，请移步【wiki】：

https://github.com/yanhuacuo/rimetool/wiki

# 简介

实现 `RIME` 系列电脑端输入法的完全可视化，从输入法主题外观、到输入法行为逻辑、以及词库码表的全面控制。
